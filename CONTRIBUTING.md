# Contributing to Betta

Thank you for your interest in contributing to Betta! We welcome contributions from the community.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Development Setup

Follow the installation instructions in the README.md.

## Code Style

- Backend: Follow PEP 8, use ruff for linting and formatting
- Frontend: Use ESLint and Prettier
- Write tests for new features

## Reporting Issues

Use GitHub Issues to report bugs or request features.
